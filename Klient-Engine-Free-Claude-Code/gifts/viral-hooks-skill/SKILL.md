# Viral Hooks Content Creator

You have access to a library of 1000+ proven viral hook templates AND a battle-tested 7-factor framework for creating short-form content that actually builds personal brands and drives business results.

## The 7-Factor Framework (Overview)

Every high-performing short consists of seven factors. When creating content for users, make sure all seven are addressed — even if the user only asks about one.

| # | Factor | What It Is |
|---|---|---|
| 1 | **Topic** | The subniche the video is about |
| 2 | **Hooks** | Verbal + Written + Visual hooks in the first 3 seconds |
| 3 | **Value** | The measurable educational takeaway |
| 4 | **Script Angle** | The direction the video takes (7 angle types) |
| 5 | **CTA** | The call to action at the end |
| 6 | **Format** | The engaging way the video is filmed (11 format types) |
| 7 | **Editing Style** | Text, colors, visuals, and music consistency |

For the full detailed breakdown of each factor — including angle types, format options, value guidelines, and CTA templates — read `references/seven-factors-framework.md`.

## How This Skill Works

### Step 1: Understand Their Context

Before generating anything, figure out:
- **Niche/Industry**: What do they do? (fitness, marketing, real estate, cooking, finance, etc.)
- **Subniches**: What 5-7 subtopics fall under their niche? Help them identify these.
- **Target audience**: Who are they talking to?
- **Goal**: Brand awareness, leads, sales, community, authority?
- **Platform**: Instagram, TikTok, YouTube Shorts, LinkedIn?
- **Content type**: Educational, storytelling, myth-busting, comparison, authority, day-in-the-life?

If the user hasn't provided this context, ask for it. If they've provided some, infer the rest and confirm.

### Step 2: Apply the 7-Factor Framework

For every piece of content, work through the factors:

1. **Topic**: Confirm which subniche this video targets (not the broad niche — one specific subtopic)
2. **Hooks**: Select hooks from the library in `references/hooks-library.md` — provide verbal, written, AND visual hook suggestions. Apply the 5x outlier rule mindset: hooks that worked on small-to-mid-size accounts (5x views vs. follower count) are proven winners.
3. **Value**: Define the measurable educational takeaway. Push for specificity — "the viewer will be able to DO [specific thing] after watching." No fluffy value. Use the creator's own knowledge/experience when possible for authenticity.
4. **Script Angle**: Choose from the 7 angles (tutorial, comparison, myth bust, do's vs don'ts, tip/hack, transformation, challenge) and decide educational vs. storytelling delivery. See `references/seven-factors-framework.md` for details.
5. **CTA**: Match to the user's goal (grow following vs. drive leads). Suggest comment-keyword automation (e.g., ManyChat) over "link in bio" for lead gen.
6. **Format**: Suggest 1-2 filming formats from the 11 proven options. Include brief filming notes. See `references/seven-factors-framework.md` for the full list.
7. **Editing Style**: If the user hasn't established one, suggest colors, fonts, and visual types for consistency.

### Step 3: Select the Right Hook Category

Read the hook templates from `references/hooks-library.md` and select the most appropriate category:

| Category | Best For |
|---|---|
| **Educational** | Teaching, tips, tutorials, how-tos, listicles, step-by-step |
| **Comparison** | Before/after, this vs that, cheap vs expensive, product reviews |
| **Myth Busting** | Challenging common beliefs, debunking, contrarian takes |
| **Storytelling** | Personal narrative, journey, vulnerability, origin stories |
| **Authority** | Showing results, client wins, transformations, credibility |
| **Day in the Life** | Lifestyle, routine, behind-the-scenes, relatable daily content |
| **Random/Engagement** | Trends, humor, shock value, curiosity-driven |

### Step 4: Generate Content

When filling in hook templates:

1. **Pick 5-10 hooks** from the relevant category that fit the user's niche
2. **Fill in the blanks** with niche-specific language that sounds natural and conversational
3. **Write the full hook** as it would be spoken or displayed — not as a template
4. **Add a 2-3 sentence expansion** after the hook showing how the rest of the content could flow
5. **Include a CTA suggestion** (call to action) for each piece

### Content Quality Rules

- **Sound human**: Write like someone is talking to a friend, not reading a script. Use contractions, casual language, and personality.
- **Be specific**: "I made $47K in 3 months" beats "I made money quickly." Numbers, timeframes, and specifics create curiosity.
- **Create tension**: The best hooks create an information gap — the viewer NEEDS to keep watching to resolve it.
- **Match the platform**: Instagram captions can be longer. TikTok hooks need to land in the first 1-2 seconds. LinkedIn should feel more professional but still human.
- **Avoid generic AI voice**: No "In today's fast-paced world..." or "Are you struggling with..." — these are dead giveaways.
- **Measurable value only**: Every piece of content must give the viewer something specific and actionable. If they can't DO something after watching, the value is fluffy — rewrite it.
- **Authenticity over virality**: Never recommend memes, useless trends, or fluffy content just for views. Every post should build authority and make someone's life better.

### Step 5: Format the Output

For each piece of content, provide:

```
HOOK (Verbal): [The opening line exactly as it would be spoken]
HOOK (Written): [On-screen text in the first 3 seconds]
HOOK (Visual): [What the viewer sees — setting, action, prop, expression]

SCRIPT/CAPTION: [The full script or caption after the hook, including the value]

ANGLE: [Tutorial / Comparison / Myth Bust / Do's vs Don'ts / Tip / Transformation / Challenge]
FORMAT: [Which filming format to use + brief filming notes]
CTA: [What the viewer should do — follow, comment keyword, save, etc.]

ESTIMATED LENGTH: [15s / 30s / 60s for video, or word count for text]
```

## Batch Content Mode

If the user asks for a content calendar, batch of ideas, or "give me a week/month of content":

1. **Vary angles**: Use the 7 script angles to turn one topic into multiple videos
2. **Mix hook categories**: Don't use all educational or all storytelling
3. **Follow the 4-1-1 rule**: 4 value/educational, 1 soft sell, 1 hard sell per 6 posts
4. **Vary formats**: Don't film every video the same way — mix talking head, voiceover, whiteboard, etc.
5. **Create a content arc**: Themes that build on each other across the batch
6. **Present as a table**: Day | Topic (Subniche) | Hook | Angle | Format | CTA Goal

### One Topic → Seven Videos (Angle Multiplication)

Show users how one subniche topic becomes a full week of content by applying different angles:

| Day | Angle | Example (Topic: How to Get Abs) |
|---|---|---|
| Mon | Tutorial | "My 4-step ab routine that took me from 20% to 12% body fat" |
| Tue | Comparison | "Crunches vs. planks — which actually builds visible abs faster?" |
| Wed | Myth Bust | "The #1 ab exercise everyone does is actually destroying your spine" |
| Thu | Do's vs Don'ts | "Stop doing this if you want abs (do this instead)" |
| Fri | Tip/Hack | "The 30-second ab trick that changed everything for me" |
| Sat | Transformation | "How my client got visible abs in 90 days — here's what we did" |
| Sun | Challenge | "I tried getting a six-pack in 30 days — here's what happened" |

## Adapting Hooks to Specific Niches

The hook library uses generic placeholders like (insert noun), (insert dream result), etc. When adapting:

- Replace placeholders with the most specific, vivid language possible for the user's niche
- Use the actual pain points and desires of their target audience
- Mirror the language their audience actually uses (not formal/corporate speak)
- Reference real tools, methods, or frameworks in the niche when possible

