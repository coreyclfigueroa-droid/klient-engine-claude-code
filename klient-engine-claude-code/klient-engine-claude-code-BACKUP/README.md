# Klient-Engine Free Claude Code Course

Free interactive course that runs inside Claude Code. Go from zero to a fully personalized AI system for your business in under an hour.



---

## Step 1 — Install Claude Code

Open your terminal and paste the command for your system:

**macOS / Linux / WSL:**
```
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows (PowerShell):**
```
irm https://claude.ai/install.ps1 | iex
```

*On Mac: search "Terminal" in Spotlight. On Windows: open PowerShell.*

---

## Step 2 — Install the Course

**macOS / Linux / WSL:**
```
curl -fsSL https://raw.githubusercontent.com/klient-engine/Klient-Engine-Free-Claude-Code/main/install.sh | bash
```

**Windows (PowerShell):**
```
irm https://raw.githubusercontent.com/klient-engine/Klient-Engine-Free-Claude-Code/main/install.ps1 | iex
```

This downloads all 5 lessons, skills, and gifts to your machine.

---

## Step 3 — Start the Course

Open Claude Code:

```
claude
```

Then type:

```
/klient-engine:start
```

**That's it. Claude takes over from here.**
